# remote-desktop-ts

터미널에서 한 줄 치면 내 컴퓨터 화면을 폰이나 다른 PC 브라우저로 볼 수 있다.
마우스, 키보드, 터치 다 된다. 포트포워딩 없고, 도메인 살 필요 없고, 계정도 필요 없다.

```
npx remote-desktop-ts --password 1234
```

끝. URL 하나 뜨면 그걸로 접속하면 된다.

---

## 어떻게 동작하는 건지

```
내 노트북                    Cloudflare (무료)              내 폰/다른 PC
[화면 캡처] → [서버] → [cloudflared 터널] → [https://xxx.trycloudflare.com] → [브라우저]
```

- Cloudflare의 무료 터널을 써서 외부에서 접속 가능한 HTTPS URL을 자동으로 만들어준다
- `cloudflared` 바이너리도 없으면 알아서 다운받는다
- 화면은 JPEG으로 압축해서 WebSocket으로 쏜다
- 마우스/키보드 입력은 반대로 WebSocket → 서버 → OS 명령으로 실행

---

## 설치 & 실행

Node.js 18+ 필요.

```bash
# 바로 실행 (설치 없이)
npx remote-desktop-ts --password 내비밀번호

# 또는 글로벌 설치
npm install -g remote-desktop-ts
remote-desktop-ts --password 내비밀번호
```

## 옵션

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| `--password` | (필수) | 접속 비밀번호 |
| `--port` | 3000 | 로컬 서버 포트 |
| `--fps` | 15 | 초당 프레임 수 (브라우저에서도 조절 가능) |
| `--quality` | 75 | JPEG 품질 1~100 |
| `--scale` | 0.8 | 화면 축소 비율 0.1~1.0 |

```bash
# 예: 느린 네트워크에서
remote-desktop-ts --password 1234 --fps 10 --quality 50 --scale 0.5
```

## 브라우저에서 할 수 있는 것들

### PC 브라우저
- 마우스 클릭, 드래그, 스크롤 — 그냥 쓰면 된다
- 키보드 입력 — 그냥 치면 된다
- `i` 키 — 상태바(FPS, CPU, 메모리) 토글

### 모바일 브라우저

터치 기기면 하단에 툴바가 자동으로 뜬다.

| 동작 | 방법 |
|------|------|
| 클릭 | 탭 |
| 드래그 | 길게 터치 후 끌기 |
| 우클릭 | 길게 누르기 (0.6초) 또는 "우클릭" 버튼 |
| 스크롤 | 두 손가락 위아래 스와이프 |
| 줌 | 두 손가락 핀치 (터치한 곳 기준으로 확대) |
| 줌 리셋 | 세 손가락 탭 또는 "줌 리셋" 버튼 |
| 키보드 | "⌨ 키보드" 버튼 |
| 데스크톱 전환 | "앱 전환" → 앱 선택하면 해당 화면으로 이동 |
| 모니터 전환 | "모니터" → 원하는 모니터 선택 |
| 수식키 | Ctrl, Alt, ⌘, ⇧ 버튼 (한번 탭 → 다음 입력에 적용) |

### 앱 전환

"앱 전환" 버튼을 누르면 지금 실행 중인 앱 목록이 뜬다.
윈도우가 여러 개인 앱은 탭하면 윈도우 목록이 펼쳐져서 원하는 걸 선택할 수 있다.

선택하면:
1. 해당 앱(윈도우)에 포커스
2. 그 앱이 있는 모니터로 화면 자동 전환

### 모니터 전환

멀티 모니터 환경이면 "모니터" 버튼으로 보고 싶은 모니터를 선택할 수 있다.

### 종료

브라우저 왼쪽 상단 "종료" 버튼 → 확인 → 서버 프로세스 종료.
비밀번호를 아는 사람만 종료할 수 있다.

## macOS에서 처음 실행할 때

두 가지 권한이 필요하다. 없으면 안내가 뜨니까 따라하면 된다.

1. **화면 녹화** — 시스템 설정 → 개인정보 보호 → 화면 녹화 → 터미널 체크
2. **접근성** — 시스템 설정 → 개인정보 보호 → 접근성 → 터미널 체크

권한 설정 후 터미널 재시작 필요.

## 알아둘 것

- URL은 실행할 때마다 바뀐다 (Cloudflare 무료 티어 특성)
- 프로세스 켜놓은 동안에는 URL 안 바뀐다
- 동시 접속은 1명만 가능 (새 접속 시 기존 연결 끊김)
- 네트워크 느리면 FPS/품질이 자동으로 낮아진다 (브라우저에서 수동 조절도 가능)
- Retina 디스플레이 좌표 보정 자동 처리
